# Note Board

## Description

A modern, highly interactive sticky notes board built with React, TypeScript, TanStack Router, TanStack Query, Supabase, and a Vercel Serverless API.
The app supports draggable and resizable markdown notes, folder organization, color customization, and integrated AI tools for generating, restyling, and expanding note content.

## Features

1. **Rich Notes:**

- Create, edit, and delete sticky notes.
- Markdown editor with preview mode.
- Resize notes via bottom-right drag handle.
- Drag notes anywhere with precise cursor-follow.
- Change note's background and text color.

2. **AI Features:**

- Generate Note from Prompt (Enter a prompt and get a full markdown note instantly).
- Restyle Existing Note Content (Convert your note to a new style - structured, bullet points, more formal, simplified, etc).
- Expand Note Content (AI expands the current note with more details or context).

3. **Folders & Organization**

- Sidebar listing all folders.
- Folder creation with automatic name conflict handling (New Folder, New Folder 2, New Folder 3, ...)
- Filter notes by folder instantly.
